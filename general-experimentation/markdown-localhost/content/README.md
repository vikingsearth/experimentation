# Markdown Localhost Showcase

Welcome to the **Markdown Localhost** experiment. This site is built entirely from
plain Markdown files, rendered as beautiful web pages in your browser using
[Docsify](https://docsify.js.org).

## What You Are Looking At

This is a live demonstration of how Markdown files can be served as a polished
documentation site on `localhost` with:

- **No build step** -- Markdown is rendered directly in the browser
- **Sidebar navigation** -- see the links on the left
- **Full-text search** -- try the search box at the top
- **Syntax highlighting** -- code blocks with language-aware coloring
- **Copy-to-clipboard** -- hover over any code block to copy

## Pages in This Showcase

| Page | What It Demonstrates |
|------|----------------------|
| [Markdown Features](features.md) | Headers, lists, emphasis, links, images |
| [Code & Tables](code-and-tables.md) | Fenced code blocks, syntax highlighting, tables |
| [Advanced](advanced.md) | Blockquotes, task lists, horizontal rules, HTML |

## Try Editing

Open any `.md` file in the `content/` directory, make a change, and refresh your
browser. The changes appear instantly -- no rebuild needed.
